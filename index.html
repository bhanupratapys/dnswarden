Features

DnsWarden provides two types of DNS:

Pure uncensored DNS (Blocks nothing, more power for the user!).
Adblocking DNS (Blocks ads, trackers and all the random crap on the internet).
If your favourite website gets blocked/broken, report here!

Supports DNSCrypt, DNS over HTTPS, DNS over TLS and Normal DNS (plaintext/cleartext).
DNSSEC and QNAME minimization are enabled by default.
Supports OpenNIC TLD's.
EDNS Client Subnet (ECS) is disabled.
No query logging.

FAQ

What's your privacy policy and T&C?

1. Use DnsWarden at your own risk. I'm not responsible for any downtime of the servers, only on a "best effort" basis.
2. No query logging. Completely disabled query logs related to the DNS traffic.

Where can I check Health/DNS Query Rate Graph?

1. Live at. The graph what you see here is equal to "Total number of queries received by servers"
minus "Cache hits", because each backend has its own resolver with cache enabled.
So it is kinda hard to know the "Total numbers of queries received by the servers"
without disabling the resolver cache on the backends.
2. Uptimerobot
